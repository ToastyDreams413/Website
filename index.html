Github

Monte Carlo Tree Search Visualization

Sole Developer

An interactive web demo that visualizes the Monte Carlo Tree Search algorithm, showing how decisions emerge through simulation and backpropagation.

Live Demo Github

I built this project to better understand and explain Monte Carlo Tree Search (MCTS), which is an algorithm commonly used in game AI and planning. The demo visualizes selection, expansion, simulation (rollouts), and backpropagation, showing how visit counts and value estimates evolve as the tree grows.

The UI is designed for clarity: you can see node values, visit counts, and the decision path that the search currently favors.
